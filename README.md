 Contact Management System (Python)

## Description
The Contact Management System is a Python-based application that allows users to manage their contacts efficiently. Users can add, view, search, update, and delete contact information. The system uses file handling to store data permanently.

## Features
- Add new contacts (Name, Phone Number, Email)
- View all saved contacts
- Search contacts by name or phone number
- Update existing contact details
- Delete contacts
- Data persistence using file handling (TXT / CSV)

## Technologies Used
- Python
- File Handling
- Functions
- Loops
- Conditional Statements

## How to Run the Project
1. Clone or download the repository
2. Open the project folder
3. Run the Python file using the command:
   ```bash
   python contact_management.py
